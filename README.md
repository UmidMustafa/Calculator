# Calculator Project

A simple command-line calculator written in Python. This project allows users to perform basic and some advanced mathematical operations through an easy-to-use text interface.

## Features

The calculator supports the following operations:
1. Addition
2. Subtraction
3. Multiplication
4. Division
5. Square Root
6. Power (Exponentiation)
7. Factorial
8. Modulo



## Architecture

The project consists of two main files:
- `main_calc.py`: The entry point of the application. It handles the user interface, taking input for operations and numbers, and printing the results.
- `calculation.py`: Contains the `Calculator` class which encapsulates all the mathematical logic and operations.
